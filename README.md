# Gestão de Loja - Da Linha de Comando à API REST

Este repositório registra a evolução da minha jornada no desenvolvimento de software, demonstrando a transição de um sistema local em linha de comando para uma arquitetura web moderna.

## Estrutura do Repositório

*   **`system-storage-java/`**: Primeira versão do sistema. Uma aplicação baseada em console (CLI) desenvolvida para validação das regras de negócio, fluxos de dados e lógica interna.
*   **`gestao-loja-api/`**: A evolução do projeto. Uma API REST completa e robusta construída utilizando **Spring Boot** e **Java 25**, integrada ao banco de dados relacional **MySQL** via JPA/Hibernate.

## Tecnologias e Aprendizados da API
*   **Spring Boot 4**: Configuração e estruturação do ecossistema Back-end.
*   **Spring Data JPA & Hibernate**: Mapeamento Objeto-Relacional (ORM) e automação de queries.
*   **MySQL**: Modelagem de dados, chaves primárias e persistência robusta.
*   **Postman**: Testes automatizados e validação de requisições HTTP (GET, POST).
